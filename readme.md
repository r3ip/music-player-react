# Reproducor de música - React

Este es un proyecto de practica utilizando React y tailwindcss

## Dependancias usadas

  - [React]
  - [Axios]
  - [Tailwindcss]
  - [React icons]

## Instalacion

```
npm i o npm install
```

## Plugins de tailwindcss

 - [DaisyUi]

## Ejecución del proyecto en local

```
npm run dev
```
